# workstreams

此目錄為 stage 功能分流骨架，每個 workstream 固定包含：

- `plan.md`
- `task.md`
- `todo.md`

## writing-plans 對齊要求

`plan.md` 至少包含以下段落：

- `## Scope`
- `## Steps`
- `## Relevant files`
- `## Verification`
- `## Decisions`

`todo.md` 至少包含以下段落：

- `## Current Sprint`
- `## Blockers`
- `## Evidence / Links`
- `## Handoff Notes`

## 協作規範

1. 沒有明確寫入邊界，不得啟動平行開發。
2. 需要 sync 回 `custom-skills` 的內容，先完成該 stage 測試並附證據。
3. 共用檔（`docs/research/05...`）只允許 Stage0 分支更新。
