### Added
- **Claude Code statusline sidecar writer（補齊 footer `cc` trusted 來源）**：新增 `paulshaclaw/cost/statusline_sidecar.py`，掛在 Claude Code 的 statusLine command 上，從 stdin 擷取其 `rate_limits.{five_hour,seven_day}.{used_percentage,resets_at}` 並原子寫入 footer 讀取的 sidecar JSON（預設 `~/.agents/state/cost/claude_rate_limits.json`）——先前無人寫此檔，`cc` 一直顯示 `--`（見前一版 CHANGELOG「cdx 數值終於顯示」註記的待補項）。writer 只保留 footer 要的兩個視窗、無可用百分比時回傳 `None` 不覆蓋既有 sidecar，`main()` 吞掉所有錯誤永遠 exit 0 以免拖垮 statusline。同步讓 footer 的 `resets_at` 解析（`providers._parse_reset_value`）同時接受 ISO8601（Claude Code 的格式）與 epoch；`paulshaclaw.sample.yaml` 的 `claude` 來源補上 sidecar 形狀說明、`max_age_seconds` 與現成 writer 用法。
