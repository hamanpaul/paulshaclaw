### Added
- **`copilot-local-vllm`（co-gem）BYOK launcher 收進 repo**：原本只存在本機 `~/.local/bin`、未進版控。新增三檔補齊，對齊 `claude-gemma4` 慣例：`scripts/copilot-local-vllm`（載入 secret env → export `COPILOT_PROVIDER_*` / `COPILOT_MODEL` → exec 真正的 `copilot`，讓 Copilot CLI 走本機 vLLM 而非 GitHub 模型；內建 WSL PATH 修補與「找真 copilot、絕不遞迴自己」的解析，並在缺 `COPILOT_PROVIDER_API_KEY` 時報錯不靜默 fallback）、`scripts/install-copilot-local-vllm`（symlink 進 `~/.local/bin`，repo=runtime 免 stale drift）、`config/copilot-local-vllm.env.tmpl`（範本；host 用 RFC 5737 文件位址、key 為 placeholder，真實 env 留本機由 `.gitignore` 擋）。等同 `claude-gemma4` 的 Copilot 版，但 Copilot BYOK 原生吃 env，故不需 proxy 亦無 settings.json。
